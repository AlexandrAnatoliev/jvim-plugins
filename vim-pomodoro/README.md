</div>

`~/.vim/pack/my-plugins/start/`:


<div align="center">
  <h4>Usage</h4>
</div>

The color scheme will change after 25 minutes of Vim operation, serving as
reminder to take a break and rest.

<div align="center">
  <h4>Compatibility</h4>
</div>
 
* Vim 7.0 and above
